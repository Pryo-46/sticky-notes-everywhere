# Sticky Notes Everywhere - リリース前テストチェックリスト

  

## 概要

このドキュメントはリリース前に実施するテスト項目をまとめたものです。

  

---

  

## 1. 基本動作テスト

  

### 1.1 拡張機能の起動

- [ ] ~~拡張機能アイコンをクリックしてメニューバーが表示される~~

- [ ] ~~再度クリックでメニューバーが非表示になる~~

- [x] ページリロード後も正常に動作する

  

### 1.2 メニューバー表示モード

  

#### バーモード

- [x] 上部（top）に正しく表示される

- [x] 下部（bottom）に正しく表示される

- [x] 左側（left）に正しく表示される

- [x] 右側（right）に正しく表示される

- [x] 位置ボタンで循環切替（top → left → bottom → right → top）

  

#### フローティングモード

- [x] ドラッグで自由に移動できる

- [x] 画面外に出ないよう制限される

- [x] 位置がページリロード後も保持される

  

#### モード切替

- [x] バーモード ↔ フローティングモードが正しく切り替わる

- [x] 切替後の状態が保存される

## 2. 付箋作成テスト

- [x] ドロップ位置に付箋が作成される

- [x] 付箋の中心がドロップ位置になる

- [x] 各色（10色全て）で作成できる

## 3. 付箋操作テスト

  

### 3.1 テキスト編集

- [x] テキストを入力できる

- [x] 日本語入力が正常に動作する

- [x] 改行が正常に動作する

- [x] 長文（1000文字以上）が入力できる

- [x] 入力後1秒で自動保存される

  

### 3.2 移動（ドラッグ）

- [x] ヘッダー部分をドラッグして移動できる

- [x] 画面上端で止まる（MIN_VISIBLE_AREA: 50px）

- [x] 画面下端で止まる

- [x] 画面左端で止まる

- [x] 画面右端で止まる

- [x] 移動後の位置が保存される

  

### 3.3 リサイズ

- [x] 右下角をドラッグしてリサイズできる

- [x] 最小幅（150px）で制限される

- [x] 最小高さ（100px）で制限される

- [x] リサイズ後のサイズが保存される

  

### 3.4 重なり順（z-index）

- [x] 付箋をクリックすると最前面に表示される

- [x] 新規作成した付箋が最前面に表示される

- [x] 多数の付箋でもz-indexオーバーフローしない

  

### 3.5 ヘッダーボタン

- [x] コピーボタン：テキストがクリップボードにコピーされる

- [x] コピーボタン：1秒間ハイライト表示される

- [x] 色ボタン：カラーピッカーが開く

- [x] 色ボタン：10色から選択できる

- [x] 色ボタン：選択した色に変更される
      →👽変更してリロードすると色が戻る

- [x] 削除ボタン：付箋が削除される

- [x] 削除ボタン：ホバーで赤色表示

  

### 3.6 色とテキスト色

- [x] 明るい背景色で暗いテキスト色になる

- [x] 暗い背景色（color10: 黒）で白いテキスト色になる

## 4. 一括操作テスト

  

### 4.1 表示/非表示切替

- [x] 目玉アイコンで全付箋が非表示になる

- [x] 再度クリックで全付箋が再表示される

- [x] Alt + H ショートカットでも切り替わる

- [x] 非表示中に作成した付箋も非表示になる

  

### 4.2 全付箋コピー（エクスポート）

- [x] コピーアイコンで全付箋がクリップボードにコピーされる

- [x] Ctrl + Alt + C ショートカットでもコピーされる

- [x] コピー形式が正しい（URL、番号付きリスト）

- [x] Y座標 → X座標順でソートされる

- [x] テキストがない付箋は除外される

- [x] 成功時に緑色トーストが表示される

- [x] 付箋がない場合はエラートーストが表示される

- [x] トーストが3秒後に消える

  

### 4.3 全付箋削除

- [x] ゴミ箱アイコンで確認ダイアログが表示される

- [x] Alt + X ショートカットでも確認ダイアログが表示される

- [x] 「OK」で全付箋が削除される

- [x] 「キャンセル」で削除されない
